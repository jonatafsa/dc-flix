## 🚀 Instalando DC Flix Prisma

Para instalar o DC Flix Prisma, siga estas etapas:

Instale as dependencies:

```
npm i
```

## ☕ Usando DC Flix Prisma

Para usar DC Flix Prisma, siga estas etapas:

```
npm run dev
```
